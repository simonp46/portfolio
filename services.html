Services I offer

Frontend Development

I offer design and development of interactive user interfaces, using modern technologies such as Angular, React, and Single SPA. I focus on creating visually appealing, responsive and user-friendly experiences, ensuring that applications work well on mobile and desktop devices.

Backend Development

Design and development of server logic using C# and .NET, creating scalable and secure API's that support a variety of functionalities, such as database management, integration with external services and more. I implement security best practices and optimization to ensure efficient performance.

Made to measure

I cover the entire web application development lifecycle, from design to deployment on servers or in the cloud, using technologies such as Azure for management and monitoring. In addition, I provide ongoing support to ensure the proper functioning of applications, adjusting and improving as needed.

Chatbots Design

I design and implement chatbots solutions using tools such as Dialogflow ES-CX to automate customer service processes, integration with multiple messaging platforms and advanced personalization of conversational flows to improve user experience.

Databases

Design and implementation of relational databases with SQL Server, optimizing queries and ensuring data integrity and availability. In addition, I manage the configuration of databases in the cloud, ensuring scalability and security.

Code Refinement and Optimization

I perform code reviews and improvements to optimize performance and maintainability. I use tools such as SonarCloud to identify potential bugs and weaknesses in the code, improving its efficiency, readability and scalability.
